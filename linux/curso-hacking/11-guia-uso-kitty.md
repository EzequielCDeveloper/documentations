## Configuracion

Una vez instalado el emulador de terminal kitty, para llegar al archivo de Configuracion, la ruta usual es:
`/home/user/.config/kitty/kitty.conf`

Funcion util a agregar: `detect_urls yes`
agregar la anterior linea bash hasta el final de nuestro archivo de configuracion de la kitty.

| Atajo                                                          | Acción                                          |
| -------------------------------------------------------------- | ----------------------------------------------- |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd>          | Abrir una nueva pestaña                         |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd>              | Reabrir la última pestaña cerrada               |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>W</kbd>              | Cerrar la pestaña actual                        |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd>              | Mostrar el listado de pestañas                  |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | Cambiar entre pestañas adyacentes               |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>+</kbd>/<kbd>-</kbd> | Aumentar / disminuir el tamaño de fuente        |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>C</kbd>              | Copiar texto seleccionado                       |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>V</kbd>              | Pegar desde el portapapeles                     |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd>              | Dividir la ventana horizontalmente              |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>              | Dividir la ventana verticalmente                |
| <kbd>Alt</kbd> + <kbd>←</kbd>/<kbd>→</kbd>                     | Navegar por el historial de directorios         |
| <kbd>Ctrl</kbd> + <kbd>R</kbd>                                 | Buscar en el historial (reverse‑i‑search)       |
| <kbd>F1</kbd>                                                  | Mostrar ayuda rápida de Kitty                   |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>U</kbd>              | Convertir texto seleccionado a mayúsculas       |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd>              | Convertir texto seleccionado a minúsculas       |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd>              | Guardar la salida actual a un archivo           |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd>              | Buscar texto dentro del buffer activo           |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd>              | Marcar la línea actual como punto de referencia |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd>              | Ir a la línea marcada                           |
